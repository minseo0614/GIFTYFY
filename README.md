# GIFTYFY
> 관계와 취향 기반 선물 추천 앱


## 팀원
김현수 (카이스트 전산 22)

김민서 (이화여대 컴공 24)

## 주요 기능
### Tab 0. 회원가입 및 로그인
* 회원가입: 닉네임, 비밀번호(재확인 기능), 이름, 생일을 입력해 회원가입, 즉시 DB에 저장.
* 로그아웃을 하지 않는 한 앱을 껐다 켜도 자동으로 로그인, 위시리스트 등 설정 내용이 저장.


### Tab 1. 친구 리스트
* 다른 앱 사용자들을 리스트로 볼 수 있는 메인 화면.
* 곧 생일인 친구들을 별개의 리스트로 강조.
* 친구 정보
  * 관계 태그: 친구와의 관계를 골라 선물 추천에 반영.
  * 취향 키워드: 친구가 설정해놓은 취향 키워드 확인 가능, 선물 추천에 반영.
  * 추천 선물: 위 정보들을 바탕으로 추천 선물 3가지 표시.

### Tab 2. 선물 리스트
* 이미지, 가격, 위시리스트 추가/삭제 버튼으로 구성된 갤러리 뷰 선물 목록.
* 클릭 시 상품 상세 설명과 선물하기 버튼 제공 (네이버+ 스토어 상품 URL로 연동)


### Tab 3. 마이페이지
* 자신의 취향 키워드 설정, 친구가 나에 대한 선물 추천을 받을 때 반영.
* 캘린더: 친구의 생일이 표시되는 캘린더 제공.
* 내가 받은 선물 리스트 제공.
* 나의 위시리스트 제공.
