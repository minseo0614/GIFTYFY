# 🎁 GIFTYFY
GIFTYFY는 친구와의 관계, 취향, 위시리스트 데이터를 기반으로 맞춤형 선물을 추천하는 안드로이드 앱입니다. 추천 알고리즘을 통해 상황에 맞는 선물을 빠르게 찾고, 실제 구매까지 자연스럽게 연결할 수 있습니다.

## 📌 문제 정의
**1. 선물 선택의 어려움**

현대 사회에서는 생일, 기념일, 감사 표현 등 선물을 해야 하는 상황이 자주 발생함 하지만 상대방의 취향을 정확히 알기 어렵고, 어떤 선물이 적절한지 판단하는 데 많은 시간과 고민이 필요함 특히 관계(가족, 친구, 연인, 동료)에 따라 선물 기준이 달라져 선택 난이도가 더욱 높아짐

**2. 개인화된 추천 서비스의 부재**

기존 쇼핑 플랫폼은 인기 상품이나 가격 중심의 추천에 집중되어 있어 선물 대상자의 취향·관계·과거 선물 이력을 반영한 개인화 추천이 부족

**3. 선물 관리의 비효율성**

위시리스트, 받은 선물, 친구 생일 정보 등이 여러 앱과 메모에 분산되어 관리해 중복 선물, 기념일 누락, 구매 기록 관리의 불편함이 발생

## 🎯 프로젝트 목표
* 선물 선택 과정에서 발생하는 시간 낭비와 선택 스트레스 감소
* 관계·취향·위시리스트 데이터를 활용한 개인화 추천 경험 제공
* 위시리스트, 받은 선물, 기념일 정보를 하나의 앱에서 통합 관리
* 실제 구매 흐름(외부 쇼핑 링크)까지 자연스럽게 연결되는 서비스 구현

## 🫧 이런 분들께 추천해요!
* 🎁 선물 고를 때마다 항상 고민이 많은 분
* 🤯 상대 취향을 잘 몰라서 매번 검색만 하다 포기하는 분
* 🎂 친구 생일·기념일을 자주 놓치는 분
* 💬 관계에 맞는 선물을 고르고 싶은 분 (친구 / 연인 / 가족 / 동료)
* ❤️ 마음에 드는 상품을 위시리스트로 정리하고 싶은 분

## 📱 주요 기능

### Tab 0. 회원가입 및 로그인

<img src="images/mad_week1_tap0.gif" width="20%">

* **회원가입**
    * 닉네임(userId), 생일
    * 비밀번호, 비밀번호 재확인
    * 가입 즉시 Firebase Firestore users 컬렉션에 저장
    * 이미 존재하는 아이디는 사용 불가능
* **로그인 유지**
    * 로그아웃하지 않으면 앱을 껐다 켜도 자동 로그인
    * 로그인 상태가 유지되므로 위시리스트/설정값/기념일 기록 등이 그대로 남음

### Tab 1. 친구 리스트 (Friends)

<img src="images/mad_week1_tap1.gif" width="20%">

* **친구 전체 목록**
    * 다른 앱 사용자들을 리스트로 보여주는 메인 화면
    * 각 친구 별 기본 정보 표시: 이름, 생일
    * 관계 태그 (친구와의 관계 선택하면 선물 추천에 반영)
* **🎂 다가오는 생일**
    * 생일이 가까운 친구들을 상단에 별도 섹션으로 강조
    * “누구 선물을 먼저 준비해야 하는지”를 직관적으로 보여줌
* **친구 상세 Bottom Sheet (친구 클릭 시)**
    * 특정 친구 선택하면 바텀시트에서 더 구체적인 정보와 추천 확인 가능
    * 관계 태그 설정: 가족/친구/연인/동료/어사 중에서 선택 (선택 값은 DB에 반영되어 이후 추천 점수에 적용됨)
    * 취향 키워드 (관심사 태그) 확인: 친구가 설정한 취향 키워드를 칩 형태로 표시 (상품 태그와 비교해 추천 점수에 반영됨)
    * 위시리스트 미리보기
    * 추천 선물 미리보기
* **“선물하러 가기”**
    * 버튼을 누르면 선물 탭으로 이동하면서 그 친구를 위한 추천 모드로 전환
    * 추천 알고리즘 결과 상위 6개 상품 보여줌

### Tab 2. 선물 리스트 (Gifts)

<img src="images/mad_week1_tap2.gif" width="20%">

* **선물 목록 (갤러리 뷰)**
    * 이미지 + 상품명 + 가격 + 위시리스트 버튼으로 구성
    * 일반 모드: 전체 상품 탐색
    * 친구 추천 모드: 특정 친구에 맞춘 추천 상품 6개 노출
* **받은 선물**
    * 일반 모드: 친구 선택 후 “선물하기” 버튼을 통해 네이버 링크로 이동하면 해당 상품이 DB에 추가되어 선물 기록으로 남음
    * 친구 추천 모드: 특정 친구를 선택한 상태에서 “선물하기”를 통해 네이버 링크로 이동하면 해당 상품이 DB에 추가되어 선물 기록으로 남음
    * 받은 선물은 회색으로 표시됨
* **위시리스트 추가/삭제**
    * 상품 카드의 하트 버튼 → 추가/삭제 가능 (DB에 저장)
* **상품 상세 페이지**
    * 상품 상세 설명, 이미지, 가격 확인 가능
    * “선물하기” 버튼 → 네이버 스토어 상품 URL로 연결

### Tab 3. 마이페이지 (My Page)

<img src="images/mad_week1_tap3.gif" width="20%">

* **내 취향 키워드 설정**
    * 취향 키워드 선택 → 다른 친구가 나에게 선물 추천할 때 반영됨
* **나의 위시 리스트**
    * 내가 하트 누른 상품들이 DB 기반으로 저장되어 표시됨
* **캘린더**
    * 친구 생일이 캘린더에 표시됨
    * 기념일 추가 가능
* **내가 받은 선물 리스트**
    * 선물하기로 실제 구매 흐름이 발생한 경우 DB에 반영되어 받은 선물 리스트에 추가됨
    * 어떤 친구에게 어떤 선물 받았는지 확인 가능

## 💡 추천 알고리즘
1. **관계 점수 (Relation Score)**: 가족 / 친구 / 연인 / 동료 등 관계 태그에 따라 상품 가중치 적용
2. **취향 일치도 (Interest Match)**: 친구의 관심사 태그와 상품 태그 비교 → 일치하는 태그에 따라 점수 증가
3. **중복 방지 요소**: 이미 받은 선물, 위시리스트에 있는 상품은 점수 조정
4. **최종 점수 정렬**: 모든 상품의 점수를 계산 후 상위 N개 추천
